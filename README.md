# Sistema de Cadastro

Este projeto foi desenvolvido para a disciplina de **Engenharia de Software** e tem como objetivo implementar um sistema simples de cadastro de usuários, aplicando conceitos de versionamento com Git/GitHub, documentação e organização de projeto.

---

## Descrição do Projeto

O sistema de cadastro permite registrar, visualizar, atualizar e excluir dados de pessoas ou usuários.  
Ele serve como base para demonstrar:

- Estruturação de projeto
- Organização de requisitos
- Versionamento com Git
- Documentação técnica
- Fluxo básico CRUD (Create, Read, Update, Delete)

O sistema pode ser expandido futuramente para diferentes finalidades, como cadastro de clientes, funcionários, alunos, pacientes, etc.
